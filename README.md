# RGU Website - The RGU Way

Modernizing the RGU Website with a premium, high-end "Glassmorphic" and "Futuristic" aesthetic.

## Features
-   **Dark Mode Optimized:** A futuristic experience with multi-layered blurs and gradients.
-   **Brand Color System:** Uses RGU's signature Purple, Blue, and Green palette.
-   **Interactive Components:**
    -   Premium Navbar with brand pulse animation.
    -   Dynamic Hero Section with particle effects and rotating logo rings.
    -   Skill Passport Banner with fluid progress bars.
    -   Responsive UI components for "Global Readiness".

## Technology Stack
-   Next.js 14
-   Tailwind CSS
-   Framer Motion
-   Lucide React

## Development
```bash
npm install
npm run dev
```
